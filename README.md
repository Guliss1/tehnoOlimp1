 Бетон-тест
 
Привет! Это мой небольшой проект – интерактивный стенд для тестирования классов бетона. Здесь можно вживую посмотреть, как ведёт себя куб из бетона под нагрузкой, сравнить разные классы и выбрать подходящий для стройки. Сделано для души, но может пригодиться и на работе 

 Что умеет этот сайт
 
Анимация разрушения – 60 плавных кадров, на которых куб постепенно покрывается трещинами, разламывается и в конце эффектно разлетается 
При этом на индикаторе и прогресс-баре показывается давление в МПа – сразу видно, при какой нагрузке начинаются микротрещины, а когда всё рухнет.

Сравнение классов – выбираешь два класса (B15, B25, B90 и т.д.), и тебе выводится подробная таблица:
– прочность,
– давления для каждой стадии разрушения,
– область применения,
– разница в процентах,
– и даже совет, какой бетон лучше взять для ответственных конструкций.

Избранное – понравившиеся классы можно отметить звёздочкой , они сохранятся в браузере. Потом можно включить фильтр «только избранное» и не листать всю таблицу.

График прочности – столбчатая диаграмма всех классов, чтобы сразу оценить, кто сильнее.

Испытание нагрузкой – вводишь любое давление (например, 20 МПа), и куб показывает соответствующую стадию разрушения. Удобно для экспериментов!

Тёмная и светлая темы – переключается одной кнопкой , глаза не устают.

Русский / английский – интерфейс можно переключить на английский для иностранных коллег.

 Технологии
 
Чистый JavaScript (никаких фреймворков, только нативный код)

Canvas 2D для анимации разрушения

Chart.js для графиков

LocalStorage для хранения избранного

CSS-переменные для смены темы

Медиазапросы для мобильных устройств
